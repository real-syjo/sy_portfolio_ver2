Spring FrameWork

미술, 명상, 영상, 감정 4가지의 주제로 심리치료를 목적으로하는 웹사이트 입니다. 저는 명상치료 기능을 맡아 테마별 명상음악 치료 기능을 구현하였습니다. 그리고 치료후기를 등록하고 공유할 수 있도록 했습니다. 치료후기 게시판은 단순한 정보공유에 더하여 좋아요 기능과 신고기능을 통해 게시물의 관리가 가능하도록 구현했습니다.

구현 기능

명상 치료

명상치료가 필요한 사용자에게 제공되는 명상 음악 재생 서비스를 제공하는 기능입니다. 관리자 페이지를 따로 두어 오디오 파일을 업로드 하고 삭제 및 조회할 수 있는 기능을 구현하였습니다.

자바스크립트를 통한 audio 객체 기능 활용에 대해서 배울 수 있었습니다. 관리자의 페이지를 따로 두어 파일을 업로드하고 관리하는 기능을 만들기 위해 관리자로써 어떤 것을 고려해야 하는지를 고민해 볼 수 있었습니다.

블랙리스트 관리

블랙리스트 기능은 사용자가 치료 후기 게시판에서 불건전한 게시물에 대한 신고를 할 수 있는 기능입니다. 관리자가 블랙리스트를 관리할 수 있도록 블랙리스트 관리 페이지를 별도로 구현했습니다.

블랙리스트의 제한 규정에 대해서 생각해보고 사용자의 웹 어플리케이션 이용제한 및 권한에 대해서 생각해 볼 수 있는 기회였습니다. 계정별 신고 count가 3회가 되면 자동으로 사용자 계정이 비활성화되며 비활성화 시 해당 계정을 페이지 접근이 불가하도록 하였습니다. 게시물의 key값, 사용자ID, 계정별 신고된 count정보,활정화 비활성화 상태값 정보등을 수집할 수 있도록 테이블을 설계하여 데이터 관리가 가능 하도록 하였습니다.

치료후기 게시판

CRUD가 가능한 상담후기 게시판입니다. 상담후기에 대해 사용자들이 정보를 한눈에 공유하고 좋아하는 게시물에 좋아요를 표시할 수 있는 기능을 구현하였습니다.

사용자 중심의 UI/UX에 대해 고려해볼 수 있는 경험이였습니다. 부트스트랩의 모달창과 슬라이더를 적용으로 클릭 수를 줄여한 편의성을 높힌 기능을 구현하였습니다. 좋아요 기능을 통해 스크립트를 많이 다뤄볼 수 있었습니다. DB 테이블을 따로 설계하여 해당 게시물의 key값, 사용자ID 그리고 게시물의 좋아요 체크 카운트 정보등을 수집하는 테이블을 설계해보는 경험을 할 수 있었습니다.
